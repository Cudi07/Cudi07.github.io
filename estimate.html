Project Overview

The Estimate App is a comprehensive financial estimation tool designed for users to compute their required savings for a time period in order to purchase a certain product.

Technologies Used

Java Firebase

Key Features

Dynamic Calculations

Real-time calculation of total as items are added or modified.

Support for Various Products

Wide selection of products to create a goal for.

User History

Users can create accounts to help with tracking.

Item Database

Reusable product/service database with history and quick-add functionality.

Mobile Base Desgin

Fully functional on mobile devices.

Outcome & Results

The Estimate App has been adopted by over 2 million users since its launch, with users reporting an average 40% reduction in time spent creating estimate.
